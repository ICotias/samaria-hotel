# Samaria Hotel

Landing page do Hotel Samaria — Irecê, Bahia.
Site estático (HTML/CSS/JS) com modal de reserva que envia os dados via WhatsApp.

## Stack
- HTML, CSS e JavaScript puro (single-file, sem build)
- Deploy na Vercel (`vercel.json`)

## Rodar localmente (yarn)
```bash
yarn install
yarn dev
```
Acesse http://localhost:8080
